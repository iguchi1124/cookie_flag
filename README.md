# CookieFlag

Cookie based feature flags implementation for Rails.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'cookie_flag'
```

And then execute:

```shell
$ bundle
$ bundle exec rails generate cookie_flag:install
```
